What are Pokémon?

Pokémon are creatures of all shapes and sizes who live in the wild or alongside humans. For the most part, Pokémon do not speak except to utter their names. There are currently more than 700 creatures that inhabit the Pokémon universe.

Types of Pokémon

There are eleven main energy type of Pokémon:

Grass

- Grass type Pokémon are super effective against: Water, Ground, and Rock type Pokémon.

- Grass type Pokémon are vulnerable to: Ice, Fire, Poison, Bug, and Flying type Pokémon.

- Grass type Pokémon aren't very effective against: Fire, Poison, Bug, Flying, and Steel type Pokémon.

Fire

- Fire type Pokémon are super effective against: Ice, Steel, Bug, and Grass type Pokémon.

- Fire type Pokémon are vulnerable to: Water, Ground, and Rock type Pokémon.

- Fire type Pokémon aren't very effective against: Water, Rock, and Dragon type Pokémon.

Water

- Water type Pokémon are super effective against: Ground, Fire, and Rock type Pokémon.

- Water type Pokémon are vulnerable to: Grass and Electric type Pokémon.

- Water type Pokémon aren't very effective against: Grass and Dragon type Pokémon.

Electric

- Electric type Pokémon are super effective against: Flying and Water type Pokémon.

- Electric type Pokémon are vulnerable to: Ground type Pokémon. In fact they are completely ineffective against Ground type Pokémon.

- Electric type Pokémon aren't very effective against: Grass type Pokémon.

Psychic

- Psychic type Pokémon are super effective against: Fighting and Poison type Pokémon.

- Psychic type Pokémon are vulnerable to: Dark, Bug, and Ghost type Pokémon. In fact they are completely ineffective against Dark type Pokémon.

- Psychic type Pokémon aren't very effective against: Steel type Pokémon.

Fighting

- Fighting type Pokémon are super effective against: Steel, Ice, Dark, Rock, and Normal type Pokémon.

- Fighting type Pokémon are vulnerable to: Psychic, Flying, and Fairy type Pokémon. Although they aren't vulnerable to Ghost type Poké, Fighting type Pokémon are completely ineffective against Dark type Pokémon.

- Fighting type Pokémon aren't very effective against: Psychic, Bug, Flying, Poison, and Fairy type Pokémon.

Dark

- Dark type Pokémon are super effective against: Psychic and Ghost type Pokémon. In fact they are completely resistant to Psychic type Pokémon

- Dark type Pokémon are vulnerable to: Fighting, Bug, and Fairy type Pokémon.

- Dark type Pokémon aren't very effective against: Fighting and Fairy type Pokémon.

Steel

- Steel type Pokémon are super effective against: Ice, Fairy, and Rock type Pokémon. In fact they are completely resistant to Poison type Pokémon.

- Steel type Pokémon are vulnerable to: Fighting, Fire, and Ground type Pokémon. Although they aren't vulnerable to Poison type Poké, Steel type Pokémon are completely ineffective against Poison type Pokémon.

- Steel type Pokémon aren't very effective against: Steel type Pokémon.

Fairy

- Fairy type Pokémon are super effective against: Fighting, Dark, and Dragon type Pokémon. In fact they are almost completely resistant to Dragon type Pokémon.

- Fairy type Pokémon are vulnerable to: Steel and Poison type Pokémon.

- Fairy type Pokémon aren't very effective against: Steel, Poison, and Fire type Pokémon.

Dragon

- Dragon type Pokémon are super effective against: Dragon type Pokémon.

- Dragon type Pokémon are vulnerable to: Steel and Fairy type Pokémon.

- Dragon type Pokémon aren't very effective against: Dragon type Pokémon.

Normal

- Normal type Pokémon are super effective against: This will vary from Pokémon to Pokémon amongst the Normal type Pokémon.

- Normal type Pokémon are vulnerable to: This varies from Pokémon to Pokémon amongst the Normal type Pokémon.

- Normal type Pokémon aren't very effective against: Again, this varies from Pokémon to Pokémon amongst the Normal type Pokémon.
